JavaScript Basics

How does JavaScript (JS) compare to HTML and CSS?

Where HTML and CSS provide the structure and design of a web page respectively, JS is responsible for interactivity on a web page. In short, JS helps a web page ‘engage’ with a user. JS can change a web page appearance based on a data derived from the way a user clicks and navigates on a web page or the information a user types into a web page.

Explain control flow and loops...

In computer programme speak, control flow is the order in which a computer executes a statement in script.

For us Coding 101 beginners (yes – that’s me), I find it easier to think of control flow like the process of reading a book. (Note: When I say ‘book’ – I mean the hardcover type of book. Kindle not included.) From the first line at the top of the page; to the last line at the bottom of the page, turn and repeat, that process is the control flow of reading a book.

A loop however, is used in programming to repeatedly run a block of code – until a certain condition is met. Now, I have moments when I’m reading that I drift off and turn a page without comprehending the words. When this happens my internal ‘loop’ will make me go back and read the page until I’ve read it completely to then continue with the rest of my book.

Explain the difference between accessing data from arrays and objects...

The difference between accessing data from arrays and objects is to access data from an array an index is used, whereas either dot or brackets are used to access data (known as properties) from an object.

Most of us are taught to count from 1…2, 3 etc, zero-based indexing means you count by starting from 0…1, 2, 3. When accessing data from an array, a programmer will need to use zero-based indexing.

Let’s look at an example of an array and how to access data using zero-based indexing:

var array[16, 21, 30];

Using zero-based indexing, to access the first element from array, which has the value of 16, use:

array[0];

To access the second element which has the value of 21, use array[1]; and so on.

However, objects are variables that contain properties and are written as property:propertyvalue pairs. Let’s look at an object example:

var engie = { age: 30, lastName: “Matene”};

To access an object property you can do so by using dot (.) or bracket notations [].

s e.g: engie.age; or engie[“age”];

Explain what functions are and why are they useful...

A function is contained set of code that performs a task or calculates a value. When you define a function, know you have created a new JS command that you can call or invoke from anywhere on your page.

Functions are useful because:

  • Rather than type out the same full code command in Javascript, you only have to create the function once and from there call on it when ever you need to use it.
  • The same command can be used multiple times.
