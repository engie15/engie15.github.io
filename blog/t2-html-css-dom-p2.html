What is grid based design?

Our eyes follow patterns when we look at a page. We will look at a page left to right, then start reading down the page. For web developers, the grid based design utilizes this pattern to display/design a web page.

What's all the hype about responsive webpages?

More and more users will browse the internet on their smartphones or tablets rather than use a PC or laptop. That means we need to make sure we have responsive webpages to cater to the different screens a user would use to scroll through your website. A responsive webpage is all about the resolution. As we create webpages, we can utilise certain rules and syntax’s in HTMLand CSS where regardless of what type of device you use or even if the page is resized, it will continue to be engaging and user friendly.

What is semantic structure?

Sematic structure is the way you structure your HTML document so that it is clear and easy for a computer to read.
