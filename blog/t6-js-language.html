Blog#12 Javascript Language

Parentheses ( )

Parentheses have multiple functions that relates to functions and structures. They can be used to:

  1. Group expressions. E.g. var x = (4 – 3) * 2;
  2. Contain expressions. E.g. if (x < 20) { ...
  3. Contain a list of parameters E.g. array(x, 0, y, 100)

Brackets [ ]

Like parentheses, brackets have multiples functions as well but they differ to parentheses. Brackets can be used to:

  1. Notate/write an array. E.g. person[name, age, height]
  2. Access an array. E.g. person[0] = ‘Engie’
  3. Access an objects properties. E.g.
    var person {}
    person['name'] = 'Engie';
    person['age'] = 30;
    console.log(person['name'];
    //expected output is 'Engie'

Braces { }

Braces {} can be used to:

  1. Notate/write an object or the opening of a block statement. E.g. var person = { ‘name’ : ‘Engie Matene’}
  2. Group altogether blocks of code or statements to be executed as a group. E.g. {
    var name = ‘Engie’;
    var age = ‘30’;
    }

Single quotes '' and Double quotes ""

I’m going to answer single and double quotes together because in Javascript essentially, they mean the same thing. Single and double quotes are used to declare string values in JS code.

E.g.
var firstName = ‘Engie’;
var lastName = “Matene”;

Notice how in the firstName variable I used single quotes and the lastName variable I used double quotes? They will both work in code but the thing to remember is when using single quotes, the pair must be both single and vice versa with double quotes. You do not want to pair a single quote with a double quote.
